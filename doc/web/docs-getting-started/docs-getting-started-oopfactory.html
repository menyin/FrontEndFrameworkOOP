面向对象工厂

提供优雅、高性能、可扩展、易维护的js面向对象开发体验

定义

J面向对象工厂做了几个规约定义：

  • 类型工厂：class.js，通过该工厂可以创建简单类型，如var Shape= Class(function(config){ });。
  • 简单类型：该类型通过类型工厂创建，此类型包含基本的属性和函数定义、继承、子类调用父类函数功能。
  • 复杂类型：是在上简单类型做的扩展，即是简单类型的子类，例如在简单类型上扩展了事件、AOP、属性操作三大功能的Shape类型。
  • 业务类型：是在简单类型或者复杂类型做的扩展，即是简单类型或复杂类型的子类，例如在复杂类型上扩展了业务相关属性、函数的editResume类型。

基本使用

创建类型Simple


            var Simple = Class(function(attrsConfig){   //实例化属性赋值
                Simple.parent().call(this,
                util.merge({  //构造默认属性赋值
                   attr0:'attrVal0',
                   attr1:'attrVal1',
                   attr2:'attrVal2'
                },attrsConfig)
            ).extend(ParentClass);//继承父类型

            Simple.implement({ //类型函数构建
                func0:function(){ //类型函数定义
                   Simple.parent('func0').call(this);//调用父类型的func0函数
                },
                func0:function(){
                },
                func0:function(){
                },
            });

创建类型实例simple


            var simple = new Simple({
                attr3:'attrVal3',
                attr4:'attrVal4',
                attr5:'attrVal5'
              }
            });

使用类型实例simple


            simple.func0();
            console.log(simple.attr0);
