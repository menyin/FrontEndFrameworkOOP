特别说明：

*框架模块全部依赖模块加载器（jpjs.js）

*框架Shape类型依赖JQuery，因此其子类也依赖JQuery

*框架下的业务编码全部严格按照前端面向对象模块化模式进行

*框架下的页面路由按照模板文件路径对应，若url只到目录则对应该目录的index模板文件
